Tips to Troubleshoot

The Personal Empowerment Protocol (PEP)

  1. Try something!

    Check your spelling. Make sure that you are calling the right functions. Read what the error says, if you got an error. Use some print statememnts to check certain parts of your code is executing.

  2. Google your error!

    Copy the error given and paste on Google

  3. Rubber duck your code!

    Grab an item (water bottle, a phone, a pen, an object) and explain your code to it. Start reading line y line explaining what you mean to do on each line, and how these actions will help you achieve your goal.

  4. Ask a neighbor!

    Ask someone to have a look at your code. Make sure you can explain your neighbor what you mean on each step of the way. Pro Tip: Have a clean code!

  5. Ask a teacher!

    If none of the above worked, go talk to a teacher. Pro Tip: Make sure you have a clean code to show!
